## 分项规则：病例讨论（55分）

### 1. 病例呈现（15分）【硬证据 1.5×权重】
- **满分档（13-15分）**：病例信息分≥3 阶段递呈（如先给主诉→再给检查结果→最后给诊断），每阶段有明确信息增量。
- **中档（8-12分）**：2 阶段或递呈不够清晰。
- **低档（0-7分）**：一次性全部呈现 → 0-4 分；无病例信息 → 0 分。
- **硬证据检查**：病例信息分段标记 + 每段内容 + 递呈逻辑词（"先"/"随后"/"最后"）。

### 2. 争议点设计（15分）
- **满分档（13-15分）**：明确诊疗矛盾点 + 至少 2 种不同观点/方案 + 支持理由。
- **中档（8-12分）**：1 个争议点或观点不够充分。
- **低档（0-7分）**：无争议点或仅描述病例无讨论焦点。
- **替代方案**：允许"问题链设计"替代争议点（如问题→线索→诊断的递进链）。
- **硬证据检查**："争议/分歧/鉴别"关键词 + 具体矛盾描述 + 不同观点及理由。

### 3. 角色分配（10分）
- **满分档（8-10分）**：≥3 个角色（汇报者/质疑者/总结者/记录者），每个角色有明确任务。
- **中档（4-7分）**：2-3 个角色或任务不够具体。
- **低档（0-3分）**：1 个角色或无角色分配。
- **硬证据检查**：角色名称 + 具体任务 + 参与方式。

### 4. 思维工具（10分）
- **满分档（8-10分）**：提供具体工具（诊断树/鉴别表/流程图/思维导图），且与病例相关。
- **中档（4-7分）**：提到工具但未给出具体内容或工具与病例关联弱。
- **低档（0-3分）**：无工具或仅写"使用思维导图"无实质内容。
- **硬证据检查**：工具名称 + 具体内容/结构 + 与病例关联。

### 5. 总结要求（5分）
- **满分档（4-5分）**：明确产出要求（归纳病例特点/诊疗要点/课后任务），且可交付。
- **中档（2-3分）**：有总结但产出不够明确。
- **低档（0-1分）**：无总结或仅写"总结讨论内容"。
- **硬证据检查**：总结内容 + 产出形式 + 交付标准。
