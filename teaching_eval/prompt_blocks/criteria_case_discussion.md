## 分项规则：病例讨论（55分）

### S1 病例选择与呈现（12分）【硬证据 1.5×权重】
- **扣分锚点**：
  - 无代表性（罕见病例无教学价值说明） → 扣5分
  - 呈现不规范（一次性全部呈现，无分阶段） → 扣4分
  - 病例信息缺失关键要素（主诉/现病史/查体/辅助检查） → 扣3分
  - 无病例信息 → **该条款≤2分**
- **硬证据检查**：病例信息分段标记 + 每段内容 + 递呈逻辑词（"先"/"随后"/"最后"）。

### S2 争议点设计（12分）
- **扣分锚点**：
  - 无核心争议点 → **该条款≤4分**
  - 无讨论价值（争议过于简单或已有定论） → 扣4分
  - 仅有争议标题无展开分析 → 扣4分
  - 无不同观点/方案对比 → 扣3分
- **替代方案**：允许"问题链设计"替代争议点（如问题→线索→诊断的递进链）。
- **硬证据检查**："争议/分歧/鉴别"关键词 + 具体矛盾描述 + 不同观点及理由。

### S3 角色分配（10分）
- **扣分锚点**：
  - 无主持人/汇报者/讨论者角色 → **该条款≤4分**
  - 角色任务不明确 → 扣3分
  - 仅1个角色（如仅汇报者） → 扣4分
  - 无记录者/总结者 → 扣2分
- **硬证据检查**：角色名称 + 具体任务 + 参与方式。

### S4 思维工具（11分）
- **扣分锚点**：
  - 无诊断思路工具 → 扣4分
  - 无临床推理框架 → 扣4分
  - 提到工具但未给出具体内容 → 扣3分
  - 工具与病例关联弱 → 扣2分
- **硬证据检查**：工具名称 + 具体内容/结构 + 与病例关联。

### S5 总结要求（10分）
- **扣分锚点**：
  - 无系统性总结 → 扣4分
  - 无知识点归纳 → 扣3分
  - 无课后任务/延伸学习要求 → 扣3分
  - 仅写"总结讨论内容" → **该条款≤3分**
- **硬证据检查**：总结内容 + 产出形式 + 交付标准。
