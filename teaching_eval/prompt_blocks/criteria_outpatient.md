## 分项规则：教学门诊（55分）

### 1. 知情同意（10分）【硬证据 2.0×权重】【否决项】
- **A级（8-10分）**：书面模板/已签署；或正文明确写明告知时点、告知内容、学员参与范围、隐私保护措施。
  - 硬证据："知情同意书"关键词 + 签署/告知动作 + 时点/范围/隐私措施。
- **B级（4-7分）**：明确写明口头/书面告知安排，且有时点或流程，但无模板或隐私措施不够完整。
  - 硬证据："告知/沟通"关键词 + 口头/书面动作 + 时点/对象。
- **C级（1-3分）**：仅写"已告知/已沟通/患者同意"，内容不可复核。
  - 硬证据：只有"同意/告知"关键词，无具体动作或范围。
- **D级/模板残留（0分）【触发否决】**：完全未提及；或"见附件知情同意书"但附件为空/仅标题。
  - 硬证据检查：若正文写"见附件"但附件空白/缺失/仅标题 → 模板残留，视同未见。
- **缓冲带映射**：D级 → 红色（-40分）；模板残留 → 红色（-40分）

### 2. 接诊流程（15分）
- **满分档（13-15分）**：明确 4 个关键节点：住院医师独立接诊→评估室汇报→共同回诊室解释→病历书写与反馈，每节点有具体任务和时间。
- **中档（8-12分）**：3 个节点或任务不够具体。
- **低档（0-7分）**：≤2 个节点或仅写"学员接诊"。
- **硬证据检查**：节点名称 + 具体任务 + 时间/顺序。

### 3. 沟通指南（10分）
- **满分档（8-10分）**：提供具体话术或沟通要点模板≥3 个场景（如告知坏消息/解释检查/安抚焦虑）。
- **中档（4-7分）**：1-2 个场景或话术不够具体。
- **低档（0-3分）**：无沟通指南或仅写"注意沟通技巧"。
- **硬证据检查**：场景名称 + 具体话术/要点 + 适用情境。

### 4. 病历模板（10分）
- **满分档（8-10分）**：提供 SOAP 病历模板或门诊病历示例，结构完整（S主观/O客观/A评估/P计划），且有填写示例。
- **中档（4-7分）**：有模板但结构不完整或无示例。
- **低档（0-3分）**：无模板或仅写"按门诊病历要求书写"。
- **硬证据检查**：模板名称 + SOAP 四要素 + 填写示例。

### 5. 反馈机制（10分）
- **满分档（8-10分）**：明确≥2 种反馈工具（Mini-CEX/DOPS/满意度问卷/阶段性反馈），且有评分标准和反馈时点。
- **中档（4-7分）**：1 种反馈工具或标准/时点不够明确。
- **低档（0-3分）**：无反馈机制或仅写"及时反馈"。
- **硬证据检查**：反馈工具名称 + 评分标准 + 反馈时点/频率。
