## 分项规则：教学门诊（55分）

### S1 知情同意（10分）【硬证据 2.0×权重】【否决项】
- **A级（8-10分）**：书面模板/已签署；或正文明确写明告知时点、告知内容、学员参与范围、隐私保护措施。
  - 硬证据："知情同意书"关键词 + 签署/告知动作 + 时点/范围/隐私措施。
- **B级（4-7分）**：明确写明口头/书面告知安排，且有时点或流程，但无模板或隐私措施不够完整。
  - 硬证据："告知/沟通"关键词 + 口头/书面动作 + 时点/对象。
- **C级（1-3分）**：仅写"已告知/已沟通/患者同意"，内容不可复核。
  - 硬证据：只有"同意/告知"关键词，无具体动作或范围。
- **D级/模板残留（0分）【触发否决】**：完全未提及；或"见附件知情同意书"但附件为空/仅标题。
  - 硬证据检查：若正文写"见附件"但附件空白/缺失/仅标题 → 模板残留，视同未见。
- **否决项触发**：D级 / 模板残留 → 否决

### S2 接诊流程设计（12分）
- **扣分锚点**：
  - 接诊步骤不规范（缺少住院医师独立接诊→评估室汇报→共同回诊室解释→病历书写与反馈任一节点） → 扣4分/节点
  - 无时间节点或时间不合理 → 扣3分
  - 各节点任务不明确 → 扣3分
  - 仅写"学员接诊" → **该条款≤4分**
- **硬证据检查**：节点名称 + 具体任务 + 时间/顺序。

### S3 沟通指南（11分）
- **扣分锚点**：
  - 无医患沟通要点 → **该条款≤4分**
  - 话术/要点<2个场景 → 扣3分
  - 无具体场景（如告知坏消息/解释检查/安抚焦虑） → 扣3分
  - 仅写"注意沟通技巧" → **该条款≤3分**
- **硬证据检查**：场景名称 + 具体话术/要点 + 适用情境。

### S4 病历模板（12分）
- **扣分锚点**：
  - 无教学病历书写规范 → **该条款≤5分**
  - 无SOAP四要素（S主观/O客观/A评估/P计划） → 扣4分
  - 无填写示例 → 扣3分
  - 仅写"按门诊病历要求书写" → **该条款≤4分**
- **硬证据检查**：模板名称 + SOAP四要素 + 填写示例。

### S5 反馈机制（10分）
- **扣分锚点**：
  - 无实时反馈 → 扣4分
  - 无课后总结 → 扣3分
  - 反馈工具<1种（Mini-CEX/DOPS/满意度问卷/阶段性反馈） → 扣3分
  - 无评分标准或反馈时点 → 扣2分
- **硬证据检查**：反馈工具名称 + 评分标准 + 反馈时点/频率。
