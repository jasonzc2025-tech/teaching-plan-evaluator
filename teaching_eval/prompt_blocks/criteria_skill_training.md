## 分项规则：技能培训（55分）

### 1. 操作标准化（15分）【硬证据 2.0×权重】
- **满分档（13-15分）**：步骤分解为≥5 个编号步骤，每个步骤有明确动作、顺序合理、关键动作完整。
- **中档（8-12分）**：3-4 个步骤或部分关键动作缺失。
- **低档（0-7分）**：≤2 个步骤或仅写"按标准操作"无具体内容。
- **硬证据检查**：步骤编号 + 动作描述 + 顺序逻辑 + 关键动作完整性。

### 2. 错误预警（15分）
- **满分档（13-15分）**：≥3 类常见错误，每类有错误表现 + 后果说明 + 预防措施。
- **中档（8-12分）**：2 类错误或后果/预防措施不够完整。
- **低档（0-7分）**：≤1 类错误或仅写"注意避免错误"。
- **硬证据检查**：错误类型名称 + 具体表现 + 后果 + 预防措施。

### 3. 评价表匹配（DOPS）（10分）【硬证据 1.5×权重】
- **满分档（8-10分）**：DOPS 评分表真实存在，评分项≥8 个（覆盖国家指南11项主要维度）且与操作步骤对应；明确采用 6 级制或类似分级标准。
- **中档（4-7分）**：DOPS 表存在但评分项<8 个或未采用分级标准。
- **低档（0-3分）**：无 DOPS 表或"见附件 DOPS"但附件空白/缺失（模板残留）。
- **硬证据检查**：DOPS 表名称 + 评分项列表（≥8项）+ 与操作步骤对应关系 + 分级标准。
- **模板残留判定**：附件缺失/空白/仅标题 → 视同未见，不得分。

### 4. 应急预案（10分）【硬证据 2.0×权重】【否决项】
- **满分档（8-10分）**：同时满足：
  1. ≥2 类异常/并发症/不良事件
  2. 每类异常≥2 项处理措施（步骤/责任人/上报转诊路径中至少两项）
  3. ≥1 项停止操作/呼叫支援/复核生命体征的触发条件
- **中档（4-7分）**：满足 2 项条件或措施不够具体。
- **低档（0-3分）**：满足≤1 项条件或仅写"如有异常立即处理"。
- **0 分/否决**：完全未提及应急预案；或"见附件"但附件空白/缺失（模板残留）。
- **硬证据检查**：异常类型名称 + 处理措施（具体步骤/责任人）+ 触发条件。
- **缓冲带映射**：
  - 红色：0 分且完全未提及/模板残留 → -40 分
  - 橙色：有预案但<2 类异常或措施不全 → -15 分
  - 黄色：满足门槛但触发条件不够具体 → -5 分

### 5. 耗材清单（5分）
- **满分档（4-5分）**：列出≥5 项耗材，含名称、规格型号、数量。
- **中档（2-3分）**：3-4 项，或缺规格/数量。
- **低档（0-1分）**：≤2 项或仅写"准备耗材"。
- **硬证据检查**：耗材名称 + 规格型号 + 数量。
