## 分项规则：技能培训（55分）

### S1 操作标准化（15分）【硬证据 2.0×权重】
- **扣分锚点**：
  - 步骤不完整（<5个编号步骤） → **该条款≤7分**
  - 无参照指南/标准来源 → 扣4分
  - 关键动作缺失 → 扣4分
  - 仅有"按标准操作"无具体内容 → **该条款≤3分**
- **硬证据检查**：步骤编号 + 动作描述 + 顺序逻辑 + 关键动作完整性。

### S2 错误预警（10分）
- **扣分锚点**：
  - 无常见错误点 → 扣4分
  - 无纠正方法/预防措施 → 扣4分
  - 错误类型<2类 → 扣3分
  - 仅写"注意避免错误" → **该条款≤3分**
- **硬证据检查**：错误类型名称 + 具体表现 + 后果 + 预防措施。

### S3 评价表匹配（DOPS）（10分）【硬证据 1.5×权重】
- **扣分锚点**：
  - 评价表与培训内容不对应 → **该条款≤4分**
  - 评分项<8个 → 扣3分
  - 未采用分级标准 → 扣3分
  - "见附件DOPS"但附件空白/缺失 → **该条款≤2分**（模板残留）
- **硬证据检查**：DOPS 表名称 + 评分项列表（≥8项）+ 与操作步骤对应关系 + 分级标准。

### S4 应急预案（10分）【硬证据 2.0×权重】【否决项】
- **满分档（8-10分）**：同时满足：
  1. ≥2 类异常/并发症/不良事件
  2. 每类异常≥2 项处理措施（步骤/责任人/上报转诊路径中至少两项）
  3. ≥1 项停止操作/呼叫支援/复核生命体征的触发条件
- **扣分锚点**：
  - 满足 2 项条件 → 扣3分
  - 满足 1 项条件 → **该条款≤4分**
  - 仅有标题无内容 → **该条款≤3分**
- **0分/否决**：完全未提及应急预案；或"见附件"但附件空白/缺失（模板残留）。
- **硬证据检查**：异常类型名称 + 处理措施（具体步骤/责任人）+ 触发条件。
- **否决项触发**：完全未提及 / 模板残留 → 否决

### S5 耗材清单（10分）
- **扣分锚点**：
  - 无耗材清单 → 扣5分
  - 清单不完整（<5项） → 扣3分
  - 缺规格型号或数量 → 扣2分
  - 仅写"准备耗材" → **该条款≤3分**
- **硬证据检查**：耗材名称 + 规格型号 + 数量。
