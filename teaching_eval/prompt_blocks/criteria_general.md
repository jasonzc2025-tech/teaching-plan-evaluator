## 类型与条款检查（通用版）

教案可能属于以下类型：

- 教学查房
- 病例讨论
- 临床小讲课
- 技能培训
- 教学门诊

请先识别实质类型 `actual_type`，再按该类型和通用条款评分：

1. 首页与基本信息完整性
2. 目标-内容-评价闭环
3. 流程设计与时间分配
4. 教学互动与反馈机制
5. 安全与规范性表达
6. 表述与格式质量

若申报类型与实质类型偏差明显，请在 `issues` 中记录，并在 `summary.actual_type` 给出识别结果。

## 类型识别关键词（用于 TCI 参考）

- 教学查房：床旁查房、病历汇报、体格检查、临床思维
- 病例讨论：鉴别诊断、诊疗决策、病例分析
- 临床小讲课：知识点、理论讲解、授课、学习要点
- 技能培训：操作步骤、模拟训练、技能考核、DOPS
- 教学门诊：接诊、SOAP、Mini-CEX、门诊评估

## 通用维度评分表（45分）

| 条款 | 满分 | 扣分锚点 |
|------|------|----------|
| 首页规范性 | 4分 | 类型未标注/科室缺失/日期缺失/带教医师未标注/职称未标注，每项扣1分 |
| 教学目标 | 10分 | 使用"了解/熟悉/掌握/提高"等不可测动词 → **该条款≤5分**；目标与后文闭环<2处 → 扣3分；目标完全不可测 → **该条款≤3分** |
| 教学内容 | 10分 | 内容与类型不匹配 → 扣5分；重点不突出 → 扣3分；逻辑混乱 → 扣3分 |
| 教学设计 | 11分 | 无时间分配 → **该条款≤5分**；仅有标题无具体内容 → **该条款≤4分**；无互动设计 → 扣4分 |
| 教学评价 | 6分 | 无评价方式 → 扣3分；无反馈机制 → 扣3分 |
| 参考文献 | 4分 | 无具体文献条目 → **该条款≤2分**；核心参考文献>5年且作为核心依据 → **该条款≤1分** |
