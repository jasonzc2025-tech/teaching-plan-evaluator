## 分项规则：临床小讲课（55分）

### S1 内容聚焦度（15分）
- **扣分锚点**：
  - 主题不聚焦/知识点>3个 → **该条款≤7分**
  - 时长偏离基准>10分钟 → 扣3分
  - 无明确导入-主体-总结结构 → 扣3分
  - 明显模板残留 → 扣4分
- **硬证据检查**：课程标题 + 知识点数量 + 时长数字 + 结构比例。

### S2 互动密度（15分）
- **扣分锚点**：
  - 无具体互动问题/任务设计 → **该条款≤5分**
  - 仅写"加强互动"无具体设计 → **该条款≤3分**
  - 互动点<2个 → 扣4分
  - 互动分布不均（集中在开头或结尾） → 扣2分
- **硬证据检查**：互动类型（提问/投票/讨论/练习）+ 时机/位置 + 具体内容。

### S3 课件质量（12分）
- **扣分锚点**：
  - 无页数说明 → 扣3分
  - 无图表/影像/示意图 → 扣4分
  - 页数明显不合理（30分钟<8页或>20页） → 扣3分
  - 无课件结构说明 → 扣2分
- **硬证据检查**：页数 + 图表/影像数量 + 逻辑结构说明。

### S4 案例分层（13分）
- **扣分锚点**：
  - 无案例 → **该条款≤4分**
  - 有案例但无递进关系（简单→复杂） → 扣5分
  - 案例与主题无关 → **该条款≤3分**
  - 案例无明确学习目标 → 扣3分
- **硬证据检查**：案例数量 + 递进关系描述 + 对应学习目标。
