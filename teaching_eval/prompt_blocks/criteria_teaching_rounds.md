## 分项规则：教学查房（55分）

### 1. 查房流程（15分）
- **满分档（13-15分）**：明确写出"示教室—床旁—示教室"三部曲，且床旁时间 15-20 分钟；每阶段有具体任务描述。
- **中档（8-12分）**：有三部曲但时间未明确或阶段任务不够具体。
- **低档（0-7分）**：缺少任一阶段，或仅写"查房流程"无具体内容。
- **硬证据检查**：查找"示教室/床旁/回示教室"关键词 + 时间数字 + 各阶段任务。

### 2. 问题设计（15分）
- **满分档（13-15分）**：≥3 个关键问题，覆盖诊断/鉴别/检查/治疗，指向临床思维培养。
- **中档（8-12分）**：2 个问题或问题链不够深入。
- **低档（0-7分）**：≤1 个问题或问题过于简单/与病例无关。
- **硬证据检查**：问题编号/列表 + 临床思维关键词（诊断/鉴别/治疗）+ 与病例关联。

### 3. 教具准备（10分）
- **满分档（8-10分）**：列出≥5 项具体教具（病历、影像、听诊器、手电、血压计等），且与查房场景匹配。
- **中档（4-7分）**：列出 3-4 项，或有教具但不够具体。
- **低档（0-3分）**：≤2 项或仅写"准备教具"无清单。
- **硬证据检查**：教具名称 + 数量/规格 + 用途说明。

### 4. 分层指导（11分）
- **满分档（9-11分）**：≥2 层差异化任务（如低年资/中高年资或汇报者/观察者/总结者），每层任务明确。
- **中档（5-8分）**：2 层或任务不够具体。
- **低档（0-4分）**：1 层或仅写"分层教学"无具体内容。
- **硬证据检查**：角色/层次名称 + 具体任务 + 参与方式。

### 5. 患者知情同意（4分）【硬证据 1.5×权重】【否决项】
- **A级（4分）**：书面模板/已签署；或正文明确写明告知时点、告知内容、学生参与范围。
  - 硬证据："知情同意书"关键词 + 签署/告知动作 + 时点/范围。
- **B级（2-3分）**：明确写明口头/书面告知安排，且有时点或流程，但无模板。
  - 硬证据："告知/沟通"关键词 + 口头/书面动作 + 时点/对象。
- **C级（1分）**：仅写"已告知/已沟通/患者同意"，内容不可复核。
  - 硬证据：只有"同意/告知"关键词，无具体动作或范围。
- **D级/模板残留（0分）【触发否决】**：完全未提及；或"见附件知情同意书"但附件为空/仅标题。
  - 硬证据检查：若正文写"见附件"但附件空白/缺失/仅标题 → 模板残留，视同未见。

#### 否决项触发条件
- 知情同意 D 级（完全未提及）
- 知情同意模板残留（附件缺失或空白）
- 缓冲带映射：D级 → 红色缓冲（-40分）；模板残留 → 红色缓冲（-40分）
