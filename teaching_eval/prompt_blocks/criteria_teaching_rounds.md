## 分项规则：教学查房（55分）

### S1 查房准备（10分）
- **扣分锚点**：
  - 无病例选择依据 → 扣4分
  - 无带教方案 → 扣3分
  - 无教具准备 → 扣3分
  - 仅有标题无具体内容 → **该条款≤4分**

### S2 查房流程设计（15分）
- **扣分锚点**：
  - 三阶段结构不完整（示教室-床旁-示教室） → 扣5分
  - 无准备阶段 → 扣5分
  - 床旁时间未明确或<10分钟 → 扣3分
  - 各阶段任务不明确 → 扣3分
- **硬证据检查**：查找"示教室/床旁/回示教室"关键词 + 时间数字 + 各阶段任务。

### S3 问题设计（10分）
- **扣分锚点**：
  - 无分层提问（低年资/高年资差异） → 扣4分
  - 无开放性问题 → 扣3分
  - 问题与病例无关 → **该条款≤4分**
  - 问题过于简单（仅需回答是/否） → 扣3分
- **硬证据检查**：问题编号/列表 + 临床思维关键词（诊断/鉴别/治疗）+ 与病例关联。

### S4 分层指导（11分）
- **扣分锚点**：
  - 无针对不同年级学员的差异化教学 → **该条款≤5分**
  - 仅写"分层教学"无具体内容 → **该条款≤4分**
  - 差异化任务不明确 → 扣3分
  - 缺少角色分配（汇报者/观察者/总结者） → 扣2分
- **硬证据检查**：角色/层次名称 + 具体任务 + 参与方式。

### S5 患者知情同意（9分）【否决项】
- **A级（8-9分）**：书面模板/已签署；或正文明确写明告知时点、告知内容、学生参与范围。
  - 硬证据："知情同意书"关键词 + 签署/告知动作 + 时点/范围。
- **B级（4-7分）**：明确写明口头/书面告知安排，且有时点或流程，但无模板。
  - 硬证据："告知/沟通"关键词 + 口头/书面动作 + 时点/对象。
- **C级（1-3分）**：仅写"已告知/已沟通/患者同意"，内容不可复核。
  - 硬证据：只有"同意/告知"关键词，无具体动作或范围。
- **D级/模板残留（0分）【触发否决】**：完全未提及；或"见附件知情同意书"但附件为空/仅标题。
  - 硬证据检查：若正文写"见附件"但附件空白/缺失/仅标题 → 模板残留，视同未见。

**否决项触发条件**：
- 知情同意 D 级（完全未提及）→ 否决
- 知情同意模板残留（附件缺失或空白）→ 否决
