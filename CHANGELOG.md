# Changelog

本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/) 规范。

---

## [3.2.0] - 2026-05-01

### 新增
- 老师端新增五类教学活动写作模板与提交前自查清单，模板内容由本地固定数据提供，不增加大模型 API 消耗
- 首页新增模板与清单复制功能，便于教师粘贴到 Word 中自行完善教案
- 评审结果页新增“优先修改清单”，按必须先改、建议优化、可以完善三类聚合结构化问题
- 上传后教案信息改为可编辑表单，支持手动修正科室、带教老师、课程名称和评审轮次
- 新增 `/activity_guide` 接口，用于按教学活动类型读取本地模板和自查清单

### 变更
- 文件名自动解析结果不再作为只读文本，教师可在提交评审前修正元数据，减少后台统计偏差
- 老师端结果展示更偏向改进闭环，优先呈现下一步修改方向

---

## [3.0.0] - 2026-04-26

### 新增
- 流式实时输出（SSE）：LLM token 逐步推送至前端，评审过程可视化
- 主界面 UI 全面重设计：深蓝渐变顶栏、评分大字 KPI、结论横幅颜色区分
- 自动元数据解析：上传文件后从文件名自动识别科室、带教老师、课程名称
- 后台带教老师统计：按老师维度展示提交次数、平均分
- 后台科室统计：按科室维度展示提交次数、平均分
- 多轮评审追踪：同一课程多次提交自动记录轮次数、首次分、最高分、最近时间
- 最近记录表新增科室、老师、课程、评审轮次列
- gunicorn 改用 gevent 异步 worker，支持更高并发
- 非 root 用户运行容器，降低安全风险

### 修复
- 修复 Dockerfile 缺少 poppler-utils 导致 PDF 解析失败
- 修复 review_round 传入非数字时返回 500
- 修复恶意表名导出返回 500 改为 404
- 修复图表 canvas 高度写死导致比例异常
- 修复 Chart.js 改用国内 CDN 并加备用回退
- 修复 zipfile 文件句柄泄露
- 修复后台入口按钮在主页面直接暴露

### 安全
- SECRET_KEY 未配置时生成随机密钥并警告
- Session Cookie 加固 HttpOnly/SameSite
- Flask 直连端口绑定至 127.0.0.1
- .env 加入 .dockerignore

### 变更
- evaluate_stream 接口由一次性返回改为真正 SSE 逐 token 推送
- repository.py 新增多维统计查询函数
- admin_dashboard.html 图表动画时长压缩至 400ms
- base.html CDN 改为国内优先

## [2.6.0] - 2026-03-21

### 新增
- 百分制条款评分体系（通用维度 45 分 + 分项维度 55 分）
- 五类教案完整支持：教学查房、病例讨论、临床小讲课、技能培训、教学门诊
- 三联证据评分机制：关键词 + 具体动作/流程 + 对象/范围
- 否决项零容忍规则：知情同意、应急预案等关键安全条款缺失触发否决
- 评审防操纵机制：检测并过滤教案中植入的诱导性评分指令
- 模块化提示词架构：prompt_blocks 目录，按教案类型动态加载规则
- 研究型落库：问题分类、严重程度、条款失分、提示词版本信息
- 后台分析：类型分布、问题热度、严重程度分布、近14天趋势图
- CSV 导出：记录表、问题表、条款表
- Claude Skill 文件发布，支持本地装载批量初筛
- 开源发布至 GitHub 与 Gitee

### 变更
- 架构重构：单文件拆分为 Blueprint 模块
- 生产环境改用 gunicorn + python-dotenv
- 数据库由无持久化改为 SQLite 落库
