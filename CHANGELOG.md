# Changelog

## [3.0.0] - 2026-04-26

### 新增
- 流式实时输出（SSE）：LLM token 逐步推送至前端
- 主界面 UI 全面重设计：深蓝渐变顶栏、评分大字 KPI、结论横幅颜色区分
- 自动元数据解析：从文件名自动识别科室、带教老师、课程名称
- 后台带教老师/科室多维统计
- 多轮评审追踪：同一课程多次提交自动记录进步轨迹
- gunicorn 改用 gevent 异步 worker

### 修复
- 修复 PDF 解析缺少 poppler-utils
- 修复 review_round 非数字输入导致 500
- 修复恶意表名导出返回 500 改为 404
- 修复图表比例异常
- 修复 Chart.js 改用国内 CDN

### 安全
- SECRET_KEY 未配置时生成随机密钥并警告
- Session Cookie 加固 HttpOnly/SameSite
- Flask 直连端口绑定至 127.0.0.1
- .env 加入 .dockerignore

- 研究型后台与 CSV 导出
