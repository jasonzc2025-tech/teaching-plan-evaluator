# 教案类型识别与类型错误处理（V2.6）

## 五类教案的核心识别点

### 1. 教学查房
关键词：查房、床旁、示教室、患者、病例汇报、体格检查  
核心形态：示教室→床旁→示教室

### 2. 病例讨论
关键词：病例讨论、信息释放、争议点、诊断思路、鉴别诊断  
核心形态：围绕病例资料分步讨论，不以床旁实时接触患者为主

### 3. 临床小讲课
关键词：小讲课、授课、PPT、知识点、互动、总结  
核心形态：围绕临床问题进行短时集中讲授与互动

### 4. 技能培训
关键词：操作、步骤、演练、DOPS、并发症、耗材  
核心形态：围绕具体技能操作开展标准化训练

### 5. 教学门诊
关键词：门诊、接诊、评估室、病历书写、SOAP、患者沟通  
核心形态：住院医师参与门诊接诊全过程

---

## 常见类型错误
1. 标“教学查房”但无床旁环节 → 实质更接近病例讨论
2. 标“病例讨论”但全文为连续授课/PPT知识呈现 → 实质更接近小讲课
3. 标“小讲课”但以操作步骤和演练为主体 → 实质更接近技能培训
4. 标“教学门诊”但没有接诊流程和病历书写，只是门诊病例分析 → 实质可能更接近病例讨论

## 处理规则（V2.6）
- **仍按申报类型评分**
- 标记：
```json
{
  "类型错误": true,
  "标注类型": "教学查房",
  "实质类型": "病例讨论",
  "判断依据": [
    "无床旁环节",
    "病例为回顾性整理而非实时接触患者"
  ]
}
```

- 反馈中增加一句：
`类型错误提示：本次仍按申报类型评分，但内容实质更接近“病例讨论”，建议作者修正申报类型。`
