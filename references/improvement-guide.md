# 改进建议指南（V2.6）

## 一、通用条款改进

### 教学目标（满分10分）
- **满分要求**：类型相符 + 可观察行为 + 对象/层次明确 + 与后文至少2处闭环。
- 常见问题：
  - 只有“掌握相关知识”这类笼统目标；
  - 没写适用对象或年资；
  - 目标与流程、任务、评价脱节。
- 建议写法：
  - `低年资住院医师能够完整采集病史并归纳病例特点；高年资住院医师能够提出鉴别诊断与初步处理方案；全体学员能够在总结环节说明本次学习的关键风险点。`

### 风险评估（满分5分）
- 采用“三联证据”思路补写：
  - **关键词**：隐私/脱敏/无菌/过敏等
  - **动作/流程**：使用隔帘、退出无关人员、病例脱敏后发放、准备急救物品等
  - **对象/范围**：患者、家属、参与学员、床旁/门诊/讨论材料等
- 只写“注意隐私保护”通常不能满分。

## 二、知情同意补写建议（V2.6统一）

### A级（高档）
- 书面模板或已签署；
- 或正文明确写明：**何时告知、告知什么、学生参与什么**。

建议句式：
`教学开始前，由指导医师向患者及家属说明本次为教学查房，住院医师将参与病史采集和体格检查，告知可能增加的问诊时间及隐私保护措施，征得同意后进入床旁环节。`

### B级（中档）
- 有明确的口头/书面告知安排，且写明发生在何时或哪一步；
- 但未附模板或未写清学生参与边界。

### C级（低档）
- 仅写“已告知患者/患者同意”，不可复核。

### D级/模板残留
- 完全未提及；
- 或“见附件知情同意书”但附件为空/缺失。

## 三、临床小讲课内容聚焦改进

### 核心原则
- 主题尽量聚焦于1个临床问题；
- 知识点建议控制在1-3个；
- 目录、标题、正文要一致；
- 互动点与案例要服务于主题，不要平铺堆砌。

### 时长规则（V2.6）
- 基准30分钟；
- 每偏离1个完整5分钟区间，递增累计扣分；
- 不足1个完整5分钟区间不扣分。

### 优化建议
- 35分钟左右：删去次要扩展，保留核心问题
- 40分钟左右：压缩为3个模块，减少枝节知识
- 45分钟及以上：考虑拆分为两次小讲课或改报其他类型

## 四、真实性 / 执行质量 / 语义细节

以下问题会让条款降档：
- 只写“加强互动”“注意风险”“做好反馈”，没有具体动作；
- 只列标题，没有流程；
- 正文写有附件，附件却不存在；
- 情境与该教学活动基本形态不符。

建议修改时补齐四个问题：
1. 谁做？
2. 什么时候做？
3. 做什么？
4. 如何评价？

---

## 五、类型错误提示的写法

若教案按申报类型评分但实质更像另一类，建议反馈中加一句：
`本次仍按申报类型评分，但从结构和实施方式看，内容实质更接近××型教案，建议作者在修改时同步校正申报类型，以减少因核心条款错位造成的失分。`
