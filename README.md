# 🏥 住培临床教案质量评审 Skill

> 一个用于 Claude 的技能插件，将 Claude 转变为住院医师规范化培训（住培）临床教案的智能质控评审员。

## 功能简介

本技能依据国家《住院医师规范化培训教学活动指南》及《临床教案质量评价标准》，对五类住培教案进行**条款级、证据驱动、可复核**的百分制评分与结构化反馈。

### 支持的教案类型

| 类型 | 分项维度重点 |
|------|------------|
| 📋 教学查房 | 查房流程（三部曲）、问题设计、教具准备、分层指导 |
| 💬 病例讨论 | 病例呈现、争议点设计、角色分配、思维工具、总结要求 |
| 📖 临床小讲课 | 内容聚焦、互动密度、课件质量、案例分层、技术应用 |
| 🔧 技能培训 | 操作标准化、错误预警、DOPS匹配、应急预案、耗材清单 |
| 🏥 教学门诊 | 知情同意、接诊流程、沟通指南、病历模板、反馈机制 |

### 核心特点

- **条款驱动**：每一分必须回溯到评价标准条款 + 教案中的明确证据
- **证据先行**：先定位证据锚点，再基于证据打分，杜绝"感觉性打分"
- **可复核**：任何第三方仅凭输出即可复现评分结果
- **自动校验**：内置 V2.1 硬校验机制（否决映射、判断-得分一致性、算术校验）
- **结构化输出**：评分概览 + 条款级评分表 + 评审意见 + JSON 数据

## 安装方法

### 方式一：下载 .skill 文件（推荐）

1. 下载本仓库的 `teaching-plan-evaluator.skill` 文件
2. 打开 Claude（claude.ai 或 Claude 桌面端）
3. 进入 **设置** → **技能** → **导入技能**
4. 选择下载的 `.skill` 文件，完成安装

### 方式二：手动安装源文件

1. Clone 本仓库
2. 将 `teaching-plan-evaluator/` 整个文件夹复制到 Claude 的技能目录中

## 使用方法

安装后，在 Claude 对话中上传教案文件，然后说：

- "帮我评一下这份教案"
- "评审这个教学查房教案"
- "看看这个病例讨论教案质量如何"
- "教案质控评分"

Claude 会自动识别教案类型，按标准逐条评审，输出完整的评分报告。

### 输出示例

```
【一】评分结果概览

通用维度得分（/45）：38
分项维度得分（/55）：42
总分（/100）：80
是否触发否决项：否
评分结论：☑合格（80–89）

【二】条款级评分表
（逐条列出判断结果、得分、证据位置、简要说明）

【三】评审意见
主要优点：...
主要问题：...
改进建议：...

【四】一句话结论
...

【结构化评分数据】JSON
{ ... }

校验日志：否决映射=通过；判断-得分=通过；算术=通过；结构=通过；低分复核=未触发
```

## 评分标准来源

- 《住院医师规范化培训教学查房指南》（2021年版）
- 《住院医师规范化培训教学病例讨论指南》（2021年版）
- 《住院医师规范化培训临床小讲课指南》（2021年版）
- 《临床操作技能床旁教学指南》（2021年版）
- 《住院医师规范化培训门诊教学实施指引》（2022年版）
